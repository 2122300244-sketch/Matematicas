# Math Engineer Pro 2026

Plataforma educativa integral para el estudio de:
* **Ecuaciones Diferenciales:** Clasificación, Lineales, Exactas y Bernoulli.
* **Transformada de Laplace:** Propiedades y aplicaciones.
* **Método de Euler:** Resolución numérica de ED.

## Características
- Calculadora científica integrada.
- Quizz interactivo con 30 preguntas dinámicas.
- Biblioteca de videos tutoriales.

## Tecnologías utilizadas
- HTML5 / CSS3
- JavaScript (Vanilla)
- MathJax (para renderizado de LaTeX)
